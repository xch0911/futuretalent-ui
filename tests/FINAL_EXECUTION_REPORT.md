## 🎯 Proactive Agent 配置

### 自动触发条件

当任务满足以下**任一条件**时自动拆分：

| 条件 | 阈值 | 示例 |
|------|------|------|
| ⏱️ 时间 | > 5 分钟 | "测试整个网站" |
| 📋 步骤 | 多步骤 | "部署到生产" |
| 🧩 模块 | 多模块 | "重构用户模块" |
| 🔄 等待 | 外部资源 | "等待 API 响应" |

### 使用方法

**自动触发**（推荐）：
```
用户：帮我测试整个网站

AI 自动拆分：
1. 测试首页 (Agent-A)
2. 测试登录 (Agent-B)
3. 测试注册 (Agent-C)
4. 测试想法 (Agent-D)
5. 生成报告 (Agent-E)

并行执行，10 分钟完成
```

**手动触发**：
```bash
# 启动子代理
sessions_spawn task="任务描述" runtime="subagent"

# 查看状态
subagents action=list
```
