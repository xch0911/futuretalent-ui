## ✅ 已完成的工作

### 1. 注册功能修复

**问题 1**: JWT secret 太短
- **现象**: 注册返回 500 错误
- **原因**: HS512 算法需要至少 256 位密钥
- **修复**: 更新 `application.properties`
  ```properties
  jwt.secret=this-is-a-very-long-and-secure-secret-key...
  ```

**问题 2**: Avatar 为 null 导致 NPE
- **现象**: 注册成功但返回 500
- **原因**: `Map.of()` 不允许 null 值
- **修复**: 改用 `HashMap` 并处理 null
  ```java
  userMap.put("avatar", user.getAvatar() != null ? user.getAvatar() : "");
  ```

**状态**: ✅ 已修复并测试通过
