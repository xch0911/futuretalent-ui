## 🐛 发现的 Bug 及修复

### Bug #1: 前端统计数据写死
**问题:** 首页统计数据是硬编码的 1234/5678，不是真实数据

**修复:**
1. 后端添加 `GET /api/users/stats` 接口
2. 前端调用真实接口获取数据
3. 状态：✅ 已修复

### Bug #2: 前端无法外部访问
**问题:** Vite 默认只监听 localhost，无法从外部访问

**修复:**
1. 修改 `vite.config.ts` 添加 `host: '0.0.0.0'`
2. 或使用 `npm run dev -- --host` 启动
3. 状态：✅ 已修复

### Bug #3: 空参数查询失败
**问题:** 前端搜索时传递空字符串导致后端查询失败

**修复:**
1. 修改 `src/services/idea.ts` 清理空字符串
2. 空参数改为 undefined，不拼接到 URL
3. 状态：✅ 已修复

## 🔧 Playwright 安装问题总结

### 遇到的问题

1. **系统无 apt-get** - 无法自动安装浏览器依赖
   - **解决:** 使用系统已安装的 Google Chrome

2. **Chromium 下载失败** - 网络问题导致下载超时
   - **解决:** 配置使用系统 Chrome (`channel: 'chrome'`)

3. **测试超时** - 部分测试因网络延迟超时
   - **解决:** 调整 timeout 到 30 秒

### 成功配置

```typescript
// playwright.config.ts
export default defineConfig({
  use: {
    channel: 'chrome', // 使用系统 Chrome
    headless: true,
    baseURL: 'http://localhost:3000',
  },
  timeout: 30000,
});
```
