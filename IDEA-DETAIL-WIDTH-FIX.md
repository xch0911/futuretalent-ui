# 想法详情页宽度修复

**修复时间**: 2026-03-20 19:12  
**问题**: 想法详情页内容宽度与其他页面不一致

---

## 🐛 问题描述

### 现象
- 想法详情页宽度：900px
- 其他内容页面（FAQ/Terms/Privacy）：1200px
- 视觉上明显偏窄

### 影响
- 页面之间宽度不统一
- 内容区域显得拥挤
- 用户体验不一致

---

## ✅ 修复内容

**文件**: `src/pages/IdeaDetail/index.module.css`

**修复前**:
```css
.container {
  max-width: 900px;  /* ❌ 过窄 */
  margin: 0 auto;
  padding-top: 24px;
  padding-bottom: 48px;
}
```

**修复后**:
```css
.container {
  /* 内容宽度与其他页面保持一致 */
  max-width: 1200px;  /* ✅ 统一标准 */
  margin: 0 auto;
  padding-top: 24px;
  padding-bottom: 48px;
}
```
